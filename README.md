# React + TypeScript + Vite

## Usage
1. Установить зависимости
```shell
pnpm i
```

2. Запустить проект
```shell
pnpm dev
```

## Описание
В проекте не учитывается адаптив.

## Технологии
## Стек технологий такой:
- React 18;
- React-router-dom 6;
- Material ui 5;
- Formik + YUP;
- Typescript;
- RTK;
- Vite.
